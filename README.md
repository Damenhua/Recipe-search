# 專案名稱：forkify

## 專案介紹

Forkify 是一個基於 Web 的應用程式，允許用戶搜尋、瀏覽和保存食譜。這個專案是使用 JavaScript 和現代開發工具（如 Webpack、Babel 和 ESLint）構建的。

## 主要功能

1. **食譜搜尋**：用戶可以通過輸入關鍵字來搜尋食譜。
2. **食譜瀏覽**：用戶可以瀏覽搜尋到的食譜，查看詳細信息和成分。
3. **食譜保存**：用戶可以將喜歡的食譜保存到自己的收藏夾中。
